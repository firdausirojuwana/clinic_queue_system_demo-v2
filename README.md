## Clinic Queue System Demo

A simple Clinic Queue System with working sql database.

- **Visualizations:** Fast and flexible design side graphs with a multitude of options.
- **Dashboards:** Showing all info and statistics.
- **Explore Logs:** Sync with sql database.
- **Mixed Data Sources:** Mix different data sources in the same graph! You can specify a data source on a per-query basis. This works for even custom datasources.

## Get started

- [Get Laragon](https://laragon.org/download)
- Setup the sql database.
- Add PhPAdmin Plugin inside Laragon.
- Launch the database + PhPAdmin.
- Enjoy!
